# Starter Kit

Copy this folder to your docs directory.

## Contents

- `templates/` — Task, Concept, Reference templates
- `governance/` — Routing matrix and metadata schema

## Setup

1. Copy all files to your `docs/` folder
2. Create three directories:
   - `20-Tasks/` — How-to guides
   - `30-Concepts/` — Explanations
   - `40-Reference/` — API/fields
3. Copy templates and customize them
4. Add metadata (YAML frontmatter) to each page

## Key Rules

- One question per page
- One content type per page (Task, Concept, or Reference)
- Mandatory metadata on every page
- No content mixing

## Templates

- `Template - Task.md` — Use for procedures
- `Template - Concept.md` — Use for explanations
- `Template - Reference.md` — Use for API/fields

See root README for details.
