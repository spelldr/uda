# Glossary

Definitions of key terms and concepts.

**Format:** One term per heading

**Example:**
```markdown
## Consistency

The property that all nodes in a distributed system have the same view of data at a given point in time.

See also: Eventual Consistency, Strong Consistency
```

**Glossary terms:**
- Don't link anywhere (terminal node)
- Everything can link TO glossary
- Keep definitions concise (2-3 sentences max)
- Link to Concepts for deeper explanation
